# 数据类型
> Rust的数据类型分为两大类，分别为`标量类型(Scalar)`和`复合类型(Compound)`可以理解为单个值和数组。
> - 标量类型：`整数`,`浮点`,`布尔`,`字符`。
> - 复合类型：``

| 长度 | 有符号 | 无符号 |‎‎‎ `                                                                                                   ` |
| -----| ----- | ------ | --- |
| 8-bit  | i8 | u8 |                                                                
| 16-bit | i16 | u16
| 32-bit | i32 | u32 |
| 64-bit | i64 | u64 |
| 128-bit| i128 | u128 |
| arch   | isize | usize |
