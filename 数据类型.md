# 变量

### 声明变量
> rust默认是不可变变量，赋值后不可再重新赋值。
```rust
let some_number = 1;
println!("The value of is: {some_number}");
```
### 声明不可变变量
> 声明可变变量需要使用`mut`关键字，表示可变。
```rust
let mut some_number = 1;
some_number = 100；
println!("The value of is: {some_number}");
```

### 声明常量
> 在Rust中常量使用`const`关键字来声明并且需要标注类型，可以在任意作用域声明。\
> 常量指的是在编译时就能确定的值，常量命名规范全大写单词之间用_分隔。
```rust
const SPECIAL_NUMBER:i32 = 3;
const THREE_HOURS_IN_SECONDS: u32 = 60 * 60 * 3;
```

### 变量遮蔽(Shadowing) - Rust特性
> 同作用域中可以用与之前变量的相同的名称声明一个新变量，第一个变量被第二个变量`遮蔽`了。
```rust
let my_number = 1;
let my_number = 2; // <- 就是新建了变量，名字相同而已
```
> 下面是具体的实例
```rust
fn main() {
    let x = 1; // L1
    let x = x + 1; // L2 [1 + 1 = 2]
    {
        let x = x * 2; // G1 [2 * 2 = 4]
        println!("内部作用域的值是: {x}");
    }
    println!("X的值是: {}", x);

}
```
- 执行顺序: 看调用顺序，`L1`先被赋值然后`L2`取`L1`的值(`1`)，最终`L2`的值是`L1 + 1 = 2`，代码块`{}`它属于局部作用域，最终`G1`的值是`L2 * 2 = 4`，最后输出`G1`的`x`结果是`4`在输出`L2`的`x`结果是`2`。
- 最终结果：`4`, `2`
