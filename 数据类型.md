# 数据类型
> Rust的数据类型分为两大类，分别为`标量类型(Scalar)`和`复合类型(Compound)`可以理解为单个值和数组。
> - 标量类型：`整数`,`浮点`,`布尔`,`字符`。
> - 复合类型：``

| 长度 | 有符号 | 范围 | | 无符号 |‎‎‎ 范围 | `` |
| ----| ----- | ---- | - | ---- | ---- | --- |
| 8-bit  | $i8$ | $-128$ 到 $127$ | | $u8$ | $0$ 到 $255$ |
| 16-bit | $i16$ | $-32,768$ 到 $32,767$ | | $u16$ | $0$ 到 $65,535$ |
| 32-bit | $i32$ | $-2^{31}$ 到 $2^{31}-1$ | | $u32$ | $0$ 到 $2^{32}-1$ |
| 64-bit | $i64$ | $-2^{63}$ 到 $2^{63}-1$ | | $u64$ | $0$ 到 $2^{64}-1$ |
| 128-bit| $i128$ | $-2^{127}$ 到 $2^{127}-1$ | | $u128$ | $0$ 到 $2^{128}-1$
| arch   | $isize$ | 取决于CPU架构 | | $usize$  | 取决于 CPU 架构 |
