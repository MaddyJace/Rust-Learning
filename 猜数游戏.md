## 猜数游戏
1. 开始: 启动程序
2. 生成随机数: 程序在1到100之间生成一个随机数，并将其存储。\
   提示用户输入猜测: 程序提示用户输入一个数字，作为对随机数的猜测。
3. 判断猜测:
    - 猜对了: 如果用户的猜测与随机数相等，程序输出祝贺信息并结束游戏。
    - 猜错了：如果用户的猜测与随机数不相等，程序会判断猜测是太大了还是太小，并给出相应的提示。
4. 回到步骤3: 用户猜错了，程序会再次提示用户输入猜测，重复步骤4。

### 添加库依赖
> 开始编写前需要在`Cargo.toml`的项`[dependencies]`中添加`rand = "0.8.5"`依赖，版本随意或更高。\
> `Cargo.toml`简单概述：每个Rust项目中会有名为`Cargo.toml`的文件，它可以管理你的项目名和版本以及依赖。
```toml
[package]
name = "Study"
version = "0.1.0"
edition = "2024"

[dependencies]
rand = "0.8.5" # <- 添加依赖
```

### 代码示例
> Rust代码示例，您无需详细深入了解这个例子中的代码语法，这只是一个简单的体验！
```rust
fn main() {
    guessing_game()
}

fn guessing_game() {
    println!("猜数字小游戏");
    // 生成随机数
    // 注意: 1..=100 含义是1-99不包含100也可以写成1..100语义是一样的。
    let secret_number = rand::thread_rng().gen_range(1..=100);
    println!("秘密数字是: {}", secret_number);

    // loop 是Rust的无限循环，由break决定退出。
    loop {
        println!("请在下方输入你的猜测.");
        let mut guess = String::new();
        // 捕获输入(执行到这里时会暂停，直到用户输入为止)。
        stdin().read_line(&mut guess).expect("Failed to read line");
        // .parse是Rust的Result，OK表示获得值，Err表示错误
        let guess :u32 = match guess.trim().parse() {
            Ok(num) => num,
            Err(_) => continue,
        };

        println!("You guessed: {}", guess);
        // 分支表达式
        match guess.cmp(&secret_number) {
            // 听其名而知其意
            Ordering::Less => println!("太小了！"),
            Ordering::Greater => println!("太大了！"),
            Ordering::Equal => {
                println!("你赢了！");
                break; // <- 关键(猜对了退出循环)
            }
        }
    }
}
```
