# 变量

### 声明变量
> rust默认是不可变变量，赋值后不可再重新赋值。
```rust
let some_number = 1;
println!("The value of is: {some_number}");
```
### 声明不可变变量
> 声明可变变量需要使用`mut`关键字，表示可变。
```rust
let mut some_number = 1;
some_number = 100；
println!("The value of is: {some_number}");
```

### 声明常量
> 在Rust中常量使用`const`关键字来声明并且需要标注类型，可以在任意作用域声明。\
> 常量指的是在编译时就能确定的值，常量命名规范全大写单词之间用_分隔。
```rust
const SPECIAL_NUMBER:i32 = 3;
```
