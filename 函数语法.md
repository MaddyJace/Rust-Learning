# 函数语法
函数在 Rust 代码中非常普遍。你已经见过语言中最重要的函数之一：main 函数，它是很多程序的入口点。你也见过 fn 关键字，它用来声明新函数。
```rust
fn main() {
    println!("Hello, world!");
}
```
#### 1. 命名规范
- 函数的命名规范 全小写并用下划线分隔单词 不能以数字开头。


#### 2. 无参函数: 
```rust
fn main() {
    println!("Hello, world!");

    another_function();
}

fn another_function() {
    println!("Another function.");
}
```
> 没有任何参数和返回值的函数，通过函数名调用。

---

#### 3. 有参函数:
- 单个参数，我们在`another_function`函数中定义了因为`x`的`i32`类型的变量，可以在函数中使用！
  ```rust
  fn main() {
      another_function(5);
  }
  
  fn another_function(x: i32) {
      println!("The value of x is: {x}");
  }
  ```
- 多个参数，当定义多个参数时，使用逗号分隔，像这样：
  ```rust
    fn main() {
        print_labeled_measurement(5, 'h');
    }
    
    fn print_labeled_measurement(value: i32, unit_label: char) {
        println!("The measurement is: {value}{unit_label}");
    }
    ```

---

#### 4. 返回值函数
```rust
fn five() -> i32 {
    5
}

fn main() {
    let x = five();

    println!("The value of x is: {x}");
}
```
> 使用 `->` 声明返回类型。
