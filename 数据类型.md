# Rust数据类型
Rust 的数据类型分为两大类：**标量类型 (Scalar)** 和 **复合类型 (Compound)**。
- **标量类型**：表示单个值。包括 `整数`、`浮点`、`布尔` 和 `字符`。
- **复合类型**：将多个值组合成一个类型。包括 `元组 (Tuple)` 和 `数组 (Array)`。

---

### 1. 标量类型

#### 整数类型
变量可以不显式声明类型，Rust 默认推断为 `i32`。显式声明方式为 `let 变量名: 类型 = 值;`。

| 长度 | 有符号 | 范围 | 示例 | 无符号 | 范围 | 示例 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 8-bit | $i8$ | $-128$ 到 $127$ | `let n: i8 = -1;` | $u8$ | $0$ 到 $255$ | `let n: u8 = 1;` |
| 16-bit | $i16$ | $-32,768$ 到 $32,767$ | `let n: i16 = 1;` | $u16$ | $0$ 到 $65,535$ | `let n: u16 = 32767;` |
| 32-bit | $i32$ | $-2^{31}$ 到 $2^{31}-1$ | `let n: i32 = -1;` | $u32$ | $0$ 到 $2^{32}-1$ | `let n: u32 = 0;` |
| 64-bit | $i64$ | $-2^{63}$ 到 $2^{63}-1$ | `let n: i64 = 1;` | $u64$ | $0$ 到 $2^{64}-1$ | `let n: u64 = 100;` |
| 128-bit | $i128$ | $-2^{127}$ 到 $2^{127}-1$ | `let n: i128 = -1;` | $u128$ | $0$ 到 $2^{128}-1$ | `let n: u128 = 50;` |
| arch | $isize$ | 取决于计算机架构 | `let n: isize = 4;` | $usize$ | 取决于计算机架构 | `let n: usize = 10;` |

> **注意**：`isize` 和 `usize` 主要用于集合的索引或内存地址量。

#### 整数字面值
Rust 允许使用下划线 `_` 作为视觉分隔符以提高可读性。

| 类型 | 示例 | 描述 |
| :--- | :--- | :--- |
| 十进制 | `98_222` | 常规数字 |
| 十六进制 | `0xff` | 以 `0x` 开头 |
| 八进制 | `0o77` | 以 `0o` 开开头的数字 |
| 二进制 | `0b1111_0000` | 以 `0b` 开头 |
| 字节 (u8 only) | `b'A'` | 仅限 `u8` 类型，表示 ASCII 字符 |

#### 浮点类型
Rust 的浮点数遵循 IEEE-754 标准。默认推断类型为 `f64`。

| 长度 | 类型 | 精度 | 示例 |
| :--- | :--- | :--- | :--- |
| 32-bit | $f32$ | 单精度 | `let x: f32 = 3.0;` |
| 64-bit | $f64$ | 双精度 | `let y: f64 = 2.0;` |

#### 布尔类型
布尔类型只有两个值：`true` 和 `false`，在内存中占 1 个字节。

| 类型 | 示例 | 描述 |
| :--- | :--- | :--- |
| $bool$ | `let f: bool = false;` | 用于逻辑判断和条件控制 |

#### 字符类型
Rust 的 `char` 类型不仅是 ASCII，它代表一个 **Unicode 标量值**（4 字节），可以表示中文、日文、韩文、表情符号等。

| 类型 | 示例 | 描述 |
| :--- | :--- | :--- |
| $char$ | `let c: char = 'Z';` | **必须**使用单引号包裹 |

---

### 2. 复合类型

#### 元组 (Tuple)
元组可以将多个不同类型的值组合在一起。元组长度固定，一旦声明无法增长或收缩。

| 特性 | 示例 | 说明 |
| :--- | :--- | :--- |
| 声明 | `let tup: (i32, f64, u8) = (500, 6.4, 1);` | 括号内写类型和值 |
| 解构获取值 | `let (x, y, z) = tup;` | 通过模式匹配将元组拆分为变量 |
| 索引获取值 | `let five_hundred = tup.0;` | 使用 `.` 符号加上从 0 开始的索引 |

#### 数组 (Array)
数组中每个元素的**类型必须相同**。Rust 中的数组长度是固定的。

| 特性 | 示例 | 说明 |
| :--- | :--- | :--- |
| 声明 | `let a: [i32; 5] = [1, 2, 3, 4, 5];` | `[类型; 长度]` |
| 初始化相同值 | `let a = [3; 5];` | 等同于 `[3, 3, 3, 3, 3]` |
| 访问元素 | `let first = a[0];` | 使用索引访问，超出范围会引发 panic |

> **提示**：如果你需要一个长度可以动态变化的列表，请使用标准库提供的 `Vector`（动态数组）。
