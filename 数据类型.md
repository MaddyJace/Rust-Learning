# 数据类型
> Rust的数据类型分为两大类，分别为`标量类型(Scalar)`和`复合类型(Compound)`可以理解为单个值和数组。
> - 标量类型：`整数`,`浮点`,`布尔`,`字符`。
> - 复合类型：``


### 整数类型
> 变量可以不声明类型Rust会自动推断数据类型，显式声明类型在变量名后后面加`:类型（例如: let number:i8 = 1;）`。

| 长度 | 有符号 | 范围 | 示例 | `                   ` | 无符号 |‎‎‎ 范围 | 示例 |
| ---- | ----- | -----| --- |-| ----  | ---- | ---- |
| 8-bit  | $i8$ | $-128$ 到 $127$ | let n:i8 = -1; | | $u8$ | $0$ 到 $255$ | let n:u8 = 1; |
| 16-bit | $i16$ | $-32,768$ 到 $32,767$ | let n:i16 = 1; | | $u16$ | $0$ 到 $65,535$ | let n:u16 = 32_767; |
| 32-bit | $i32$ | $-2^{31}$ 到 $2^{31}-1$ | let n:i32 = -1; | | $u32$ | $0$ 到 $2^{32}-1$ | let n:u32 = 0; |
| 64-bit | $i64$ | $-2^{63}$ 到 $2^{63}-1$ | let n:i64 = 1; | | $u64$ | $0$ 到 $2^{64}-1$ | let n:u64 = 100; |
| 128-bit| $i128$ | $-2^{127}$ 到 $2^{127}-1$ | let n:i128 = -1; | | $u128$ | $0$ 到 $2^{128}-1$ | let n:u128 = 50; |
| arch   | $isize$ | 取决于CPU架构| $无$ | | $usize$  | 取决于 CPU 架构 | $无$ |

### 整数字面值
> 没有描述

| 数值字面值 | 示例 | `                                                                                                   ` |
| ----------| -----| - |
| 10进制 | 98_222  |
| 16进制 | 0xff   |
| 8进制  | 0b1111_0000|
| byte字节 | b'A' | b开头表示ASCII码表，无开头表示Unicode，不论怎么样都需要有`''`号包裹 |

### 浮点类型
> Rust浮点都是有符号的小数，没有显式声明浮点类型默认为`f64`双精度，跟大多数语言的`double`和`final`是一个意思`比如Java`。

| 长度 | 浮点数 | 示例 | 精度 | `                                                                               ` |
| ---- | ----- | --- |-| - |
| 32-bit  | $f32$ | let n:f32 = 1.123;   | 单精度 |
| 64-bit | $f64$  | let n:if64 = -2.345; | 双精度 |

### 字符类型
> 字符类型`let n:char = 'A';`表示一个Unicode标量值，如果要表示ASCII码的值使用`b`开头`(例如: let n:char = b'A';)`。

| 长度 | 布尔值 | 示例 | `                                                                                            ` |
| ---- | ----- | --- |-|
| 32-bit  | $char$ | let n:char = 'A'; |
